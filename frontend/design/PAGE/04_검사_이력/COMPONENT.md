# 화면 4: 검사 이력 (정밀 분석 모달 포함)

> ← [MAIN.md](../MAIN.md) | Feature: `lib/features/inspection_history/`

## 1. 개요
- **목적**: 당일 세션(1일 1세션)의 개별 촬영/분석 결과 카드 리스트. 필터링 + 정밀 분석 모달 진입점.
- **Feature 모듈**: `inspection_history` (정밀 분석 모달은 하위 Widget으로 포함)

### 핵심 개념 — 검사 이력 vs 대시보드 최근 세션 이력
```
[검사 이력 탭2]                    [대시보드 최근 세션 이력 탭1]
  당일 세션의 개별                      세션 종료된
  촬영/분석 결과 목록                  최종 결과 목록 (최근 3건)
        │                                  │
        │  "세션 완료" 클릭                │
        │  (검사 미완료 0건일 때만 가능)   │
        │  ──────────────────→             │
        │                                  │
        └──────────────────────────────────┘

※ 결과 처리 완료 건 → 검사 이력에 유지 (완료 필터에서 확인)
※ 결과 처리 미완료 건 → 검사 이력에 유지 (미완료 필터에서 확인)
※ 미완료 건이 0건이어야 "세션 완료" 버튼 활성화
```

## 6. 빈 상태 / 워크플로우 / 처리 흐름

### 6.1 카드 항목 구성
```
┌─ [빨강 바] ───────────────────────────────┐
│  [썸네일]  불량품 (Fail)       오전 10:24  │
│            균열-도장            [검사 완료]  │
│            외부 / 지지대 · 이미지 #102     │
└───────────────────────────────────────────┘

┌─ [초록 바] ───────────────────────────────┐
│  [썸네일]  양품 (Pass)         오전 10:15  │
│            표면양품-도장        [검사 완료]  │
│            외부 / 지지대 · 이미지 #045     │
└───────────────────────────────────────────┘

┌─ [회색 바] ───────────────────────────────┐
│  [썸네일]  판정 대기            오전 09:58  │
│            분석 대기 중         [검사 미완료]│
│            외부 / 지지대 · 이미지 #058     │
└───────────────────────────────────────────┘
```

### 6.2 "세션 완료" 버튼 동작 상세
- **활성화 조건**: 검사 미완료(결과 처리 미완료) 건이 0건일 때만 활성화. 1건이라도 남아 있으면 버튼 비활성화(disabled)
- **동작**:
  1. `검사_세션.세션_상태` → '완료', `종료_일시` = NOW()
  2. 세션 종료 → 대시보드 화면의 "최근 세션 이력"에 해당 세션 결과 업데이트

### 6.3 빈 상태 (EmptyState) — 필터별
세션 진입 직후(촬영 0건) 또는 필터 결과가 0건일 때 카드 리스트 자리에 다음 EmptyState 를 표시:

| 필터 | 아이콘 | 제목 | 보조 텍스트 |
|---|---|---|---|
| 전체 | `photo_camera` | "아직 촬영한 이미지가 없습니다" | "'이어서 검사'를 눌러 촬영을 시작하세요" |
| 검사 완료 | `assignment_turned_in` | "완료된 검사 결과가 없습니다" | "결과 처리가 끝난 항목이 여기에 표시됩니다" |
| 검사 미완료 | `check_circle` | "미완료 검사 결과가 없습니다" | "모든 결과가 처리되었습니다" |

아이콘 크기는 모두 48px, 색상 `on-surface-variant`.

## 8. 정밀 분석 모달 (Bottom Sheet)

### 8.1 목적
개별 불량 이미지의 상세 분석 결과 확인. 검사 이력에서 불량 카드 클릭 시 바텀 시트로 표시.

### 8.2 UI 구성
| 영역 | 요소 | 타입 | 설명 |
|---|---|---|---|
| 상단 | 드래그 핸들 | Container (12×4, rounded) | 모달 드래그 핸들 |
| 상단 | "정밀 분석 결과" + 닫기 | Row | 타이틀(h1, bold) + close 아이콘 버튼 |
| 이미지 | 원본 이미지 | InteractiveViewer | 핀치 줌 가능 (aspect 4:3, rounded) |
| 이미지 오버레이 | Grad-CAM 토글 | Switch | 반투명 배경 위 "Grad-CAM" 라벨 + ON/OFF 스위치 |
| 이미지 하단 | 줌 안내 | Text (italic, muted) | "* 이미지를 두 손가락으로 확대(Pinch Zoom)할 수 있습니다." |
| 본문 | Top-3 예측 | Card | 제목: "추론 결과 (Top-3 Predictions)" |
| Top-3 1위 | 강조 카드 | Container (border, bg) | 클래스명 + 신뢰도(%) + 프로그레스 바 (h=12px, primary-container) |
| Top-3 2위 | 일반 텍스트 | Row + ProgressBar | 클래스명 + 신뢰도(%) + 프로그레스 바 (h=10px, muted) |
| Top-3 3위 | 일반 텍스트 | Row + ProgressBar | 클래스명 + 신뢰도(%) + 프로그레스 바 (h=10px, muted) |
| 본문 | 상세 분석 데이터 | Card | 검사 유형, 신뢰도(Confidence), 최종 예측 클래스 (2열 그리드) |

| RAG 카드 | 상세 조치 | List (disc) | "[상세 조치 내역]" + 불릿 리스트 |
| RAG 카드 | 매뉴얼 출처 (Top-3) | ExpansionTile | description 아이콘 + 1순위 매뉴얼 강조 표시 (예: "표면처리 정비 지침서 - p.45 · 외 2건"). 탭하면 2~3순위 펼침 — 각 행에 매뉴얼 제목, 페이지, 청크 순서, 유사도(%) 표시 |
